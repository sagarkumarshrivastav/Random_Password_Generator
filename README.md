Modern Password Generator is a desktop application designed to provide a simple and user-friendly way to create strong, high-entropy passwords. Unlike traditional command-line scripts, this project uses the CustomTkinter library to offer a modern graphical user interface similar to Windows 11 and macOS applications.

This project is ideal for beginners and freshers who want to showcase Python GUI development skills in their portfolio.

## ‚ú® Features
1. Modern dark-mode graphical interface
2. User-defined password length
3. Generates strong passwords using:
    a. Uppercase letters
    b. Lowercase letters
    c. Numbers
    d. Special characters
4. Read-only output field for easy copying
5. Input validation with error handling

  3. Run the following command:
        python password_generator.py

How to Use
  1. Enter the desired password length (example: 12)
  2. Click on Generate Password
  3. The generated password will appear in the output field
  4. Copy and use the password securely
